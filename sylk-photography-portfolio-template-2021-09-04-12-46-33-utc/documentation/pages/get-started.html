Get started

Some good tips to get you started quickly:

  1. Never start from scratch.
  2. Use the custom.css included to make changes to your styles.
  3. Read the docs carefully before contacting support. You might be helped more quickly.
  4. If the docs don't help, use Google or stackoverflow for your specific problem.
  5. Use find and replace all, if you'd like to change more files at once.
  6. Sylk is based on Bootstrap v3.3.6. So you can use all it's handy features.
