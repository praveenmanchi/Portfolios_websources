Template Structure

To get to know what you've bought.

Sylk template

  1. index.html (overview)
  2. /css/ (all the styles)
  3. /fonts/ (font icons)
  4. /images/ (your images)
  5. /js/
    1. functions.js (template functions)
    2. vendor.js (all vendor scripts combined)
  6. /modules/
    1. tera-lightbox (Tera Lightbox 4.0) - Themanoid's exclusive
    2. titan-slider (Titan Slider 2.0) - Themanoid's exclusive
    3. img-scroller (IMG Scroller) - Themanoid's exclusive
  7. /pages/
    1. /about/ (about page)
    2. /clients/ (client page)
    3. /contact/ (contact page)
    4. /homepages/ (homepage variants)
    5. /portfolio/ (portfolio pages)
  8. /php/ (mail script)

HTML Structure

Most pages have a structure like this:

<html>
	<body> // Add your favorite helper classes here
		<div id="wrapper"> // The wrapper
			<header> 
				// Logo and main navigation
			</header> 
			<section class="x"> 
				// Section - can be a portfolio grid or content
			</section> 
		</div> 
		// Scripts
	</body>
</html>

CSS Structure

The Elements.css file has all the styled elements used in Sylk. You can modify this stylesheet at your own risk.

To adjust styles, it's better to override them using the custom.css file, also included with Sylk.
